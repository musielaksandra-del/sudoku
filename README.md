# sudoku

Projekt akademicki zrealizowany w ramach studiów. Jest to gra sudoku napisana w języku Python.

## Mój wkład w projekt
Projekt był realizowany w zespole. Do moich głównych zadań należało:
 1. Zaprojektowanie i implementacja silnika gry (logiki odpowiedzialnej za strukturę planszy sudoku.
 2. Stworzenie algorytmu sprawdzania poprawności wprowadzanych danych (walidacja wiersz, kolumn oraz sekcji 3x3 zgodnie z zasadami gry).
 3. Optymalizacja kodu pod kątem wykrywania konfliktów na planszy w czasie rzeczywistym.

## Jak uruchomić projekt
1. Upewnij się, że masz zainstalowanego Pythona.
2. Uruchom plik za pomocą komendy: `python sudoku_2.py`
